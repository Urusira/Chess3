Подробное описание

Абстрактный класс для движения, будет содержать базовые характеристики для каждого из видов движения, актуальные для каждой фигуры. Во первых это тип движения, во вторых две позиции: откуда ходим и куда ходим, а так же метод, который позволит реализовать ход, экзекьютнуть. Он будет выполнять сам себя на текущей доске
