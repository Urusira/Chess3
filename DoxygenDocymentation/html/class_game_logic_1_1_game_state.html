В этом методе мы получаем все возможные ходы для заданной позиции. В случае, если идёт попытка рассчёта для пустой клетки или фигуры цвета, не соответствующего текущему игроку, метод ничего не делает, выдавая пустоту. Если же всё в порядке, то метод получает фигуру в заданной позиции и вызывает у неё метод получения всех ходов.
